# Data Structures & Algorithms – Interview Prep

This repo is where I rebuild core data structures and algorithms from scratch as part of my interview prep. The goal is to strengthen fundamentals, understand everything at a first-principles level, and keep everything organized and clear.

## Data Structures
Implementations written without relying on high-level built-ins:

- Dynamic array  
- Linked list  
- Stack and queue  
- Circular buffer  
- Hash map  
- Heap  
- Graph  
- Trees (BST, trie, segment tree)  
- Union-Find  

Each includes a simple API, example usage, edge cases, and time/space complexity.

## Algorithms
Core algorithms grouped by topic:

- Searching and sorting  
- Backtracking and recursion  
- Two-pointers, sliding window, prefix/suffix  
- Greedy  
- Dynamic programming  
- Graph algorithms (BFS, DFS, Dijkstra, topological sort, MST, shortest paths)

## Problems
Pattern-based solutions (LC, Grokking, company sets) organized by difficulty, pattern, and company. Each solution includes the reasoning, approach, and full complexity breakdown.

## Notes
Personal notes, cheat sheets, and mind maps covering concepts like graph theory, DP templates, and complexity rules.

## Purpose
This repo reflects my approach to interview prep: clean code, strong fundamentals, and deliberate practice.

## Status
Actively updated during interview prep.
